新VAIO　黑魂革面

VAIO向為PC機貴族，當中VA兩字原意代表玩盡Video、Audio；最近舊名重新定義為更高檔VisualAudioIntelligenceOrganizer，視像觀感由流線、碳纖×鎂合金機身，到隨便配備HDMI插口、Blu-ray燒碟播片，甚至借用PS3播相睇片多媒體介面。洗心革面，VAIO新機四連發，話畀大家知AV電腦從此Out晒！

記者：朱崇德
攝影：陳陶鈞

VAIOZ　碳纖×鎂合金減磅

EeePC一類機仔係一種潮流，論全面功能、實用性，都係FullSizeNotebook高啲。VAIOZ係現時同系列頂級機，上手發現機面使用強韌碳纖，配堅固鎂合金骨架，再將主機板重新規劃減磅近4，令連光碟機、電池重量維持在1.48kg，相對市面一般都要去到1.7-1.9kg，真係相當嚇人！
機身效能冇因減磅而降低，核心使用IntelCentrino2最新技術的Core2Duo雙核心處理器，在溫度降低之餘，FSB更高達1066MHz，配合上4GBDDR3SDRAM同NVIDIAGeForce9300M顯示卡，行VistaBusiness、又播片、又上網，完全可以話爽住去。機身配備好齊全，藍芽、IEEE802.11a/b/g/n，兼內置埋燒DVD，全面功能無可挑剔。最正係佢個新設計鍵盤，同Macbook一樣使用每粒字獨立式設計，相對以往舊式一塊過設計，好處係全完聽唔到打字聲，SONY圍內工作人員仲笑稱佢做滅聲器，大家可想而知有幾勁。

Profile

IntelCore2DuoP9500（2.53GHz）、4GBRAM、320GBHDD、DVD±R/RW、13.1吋、IEEE802.11a/b/g/n、314×24.5-33×210mm、1.48kg

VAIOSR　三色平價入門

高效能唔係人人啱，VAIOSR機身顏色有粉紅、銀、黑三色選擇，睇定位明顯走實用Trendy路線。不過一樣新玩Centrino2新規格，配合DVD±R/RW燒錄光碟功能、13.3吋ClearBright闊屏靚芒，以1.88kg重量開價$9,980，以VAIO嚟講係超抵入門機。機身另備Switch功能，可以預設5種功能模式，例如喺屋企就重視多媒體配女朋友Wallpaper，返到公司就重視慳電配WindowsWallpaper，用法相當搞鬼。

Profile

IntelCore2DuoP8400（2.26GHz）、1GBRAM、200GBHDD、DVD±R/RW、13.3吋、IEEE802.11a/b/g/n、315×34×233.8mm、1.88kg

VAIOBZ　鎂合金勁好撞

睇見VAIOBZ黑實實傳統式機身，明顯係走商務路線。A4Size機度，配合鎂合金機身設計，以耐用耐撞為設計賣點，如果唔偷花巧，要保存重要資料，非常啱用。由於為公司客專設，所以機身配備有指紋辨識保安系統，配合15.4吋特大闊屏，邊打字邊上網都仲得，機價$10,980。

Profile

IntelCore2DuoP8400（2.26GHz）、2GBRAM、200GBHDD、DVD±R/RW、15.4吋、IEEE802.11a/b/g/n、362.4×29.8-37.7×266.8mm、2.7kg

VAIOFW　強玩BD睇兼燒

尤記得第一部Blu-rayNotebook開價2萬多，今日新VAIOFW系齋Play只需$14,980（VGN-FW16G/W），就算燒埋50GBBD都只係$21,980（VGN-FW17GU/H），感覺價格已經相宜好多。FW明顯預你玩靚影音，機身配備特大16.4吋、16:9闊屏，兼內置DolbySoundRoom軟件玩高規格靚聲，再加埋內置有HDMI影畫輸出Out電視，話佢係一部流動影院都不為過。
