西貢栢濤灣　海景洋房
呎價8400八個月跌16%

西貢栢濤灣是區內樓齡較新的洋房屋苑，外觀宏偉極具氣派，會所設施齊全，兼有穿梭巴士直達港鐵坑口站，相對同區大部份高齡兼設施欠奉的舊式豪宅屋苑，自有一定吸引力。不過，栢濤灣洋房實用率不足7成，加上4層高設計，層數多、上落麻煩。
栢濤灣全數52座洋房，大致分為海景及山景兩類，前者集中在雙號屋，面積分4654及5003方呎兩類；以單數為主的山景洋房，面積分4797及5003方呎兩類。另外，為數約10座的「董事屋」，一律臨海而建，私隱度較高，面積一律逾6000方呎，花園比標準洋房大，個別達3000方呎，附私家泳池，盤源罕有，今年未有成交，現時也無放盤。
盈嘉置業沈佑晉表示，栢濤灣洋房今年只有3宗成交，其中3月份易手的雙號海景洋房，呎價突破1萬元，創該屋苑歷來新高。現時海景洋房放盤，叫價最低由每方呎約8400元起，反映樓價8個月間回落至少16%，不過，相對整體樓市跌幅，抗跌力仍然較強。

全年僅3宗成交

屋苑現有12座洋房放售，海景及山景洋房平均叫價分別為每方呎9000及8000元，大部份屬原裝交樓標準，極少數連裝修盤源，叫價會稍高約5%。現時屋苑入場費約3600萬元，可購買約4800方呎的山景洋房，至於海景洋房的入場費，則要貼近4000萬元。
栢濤灣早於00年推售，其後轉租，直至04年中陸續將租約期滿單位拆售，初期洋房平均呎價約6000元，分層單位每方呎平均約5000元。
物業吸引不少名人聚居，包括名牌服飾專賣店Extravaganza老闆朱國南、英皇集團主席楊受成胞弟楊超成及前中銀香港副總裁陳子政。

五千呎屋連300萬豪裝

栢濤灣洋房前臨相思灣泳灘，在屋苑大閘已可飽覽無邊際的海景。相比之下，欠缺海景的洋房，難免有點失色，幸山景青葱翠綠。
介紹的山景洋房，面積逾5000方呎，樓高4層，1及2樓再細分兩個半層，上落樓梯多。地下入口是長形的客廳，往下走為地庫偏廳及睡房，外接約1300方呎花園。1樓兩個半層，分別是飯廳和廚房，以及附連露台的套房；2樓設另外兩個套房，包括向西北的主人套房，內附衣帽間。
洋房兩年前耗資逾300萬元改裝，至今依然甚有看頭，比原裝格局討好得多。
