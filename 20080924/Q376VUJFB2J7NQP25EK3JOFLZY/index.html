學炸高力豆沙，明明跟足食譜教路，但炸時脹卜卜，一盛起即時變樣，仲快過國技變臉，口感賣相都大打折扣！加多啲粉，皮又厚到似沙翁，究竟要點整先好味？原來只要打蛋白時加少許他他粉，就可令蛋白不易塌下，夠堅挺，而且皮薄又鬆軟！

高力朱古力

時間：15分鐘
製作難度：★★☆☆☆

不用豆沙，改用朱古力配香蕉，甜味互相中和，吃時朱古力仍保持流質，口感更好。

材料

朱古力適量、香蕉1隻、蛋白5隻、他他粉2茶匙、麵粉4湯匙、糯米粉/生粉各1/2湯匙、糖適量

做法

1. 香蕉切片，面塗上1茶匙朱古力。

2.麵粉、糯米粉及生粉混合篩勻。

3.蛋白加入他他粉，用電動打蛋器打至企身，慢慢逐少篩入（2），輕輕拌勻。

4. 用雪糕勺取半勺蛋白，下（1），再用蛋白蓋好。

5. 以大火燒熱油轉中火，從雪糕勺挖出蛋白球，放入沸油中炸至金黃及膨脹，面灑上糖伴食便可。

貼士

想高力朱古力鬆軟，麵粉及生粉不宜加太多，否則口感軟實似沙翁。

小棠菜飯

時間：30分鐘
製作難度：★☆☆☆☆

一般菜飯只加菜，這個小棠菜飯就加入鹹肉，入口有鹹香味，令人開胃。

材料

小棠菜10棵（約300克）、上湯適量、蒜頭3粒（剁茸）、生粉水適量、米/水各2杯、鹹肉5片

做法

1.米洗淨瀝亁水，放入電飯煲加水煮熟。鹹肉切碎，隔水蒸10分鐘。小棠菜洗淨瀝亁水，菜葉及莖分開切碎備用。

2. 燒熱油，以中火爆香蒜茸，下莖部略炒，注入上湯，再下葉部份炒勻，下生粉水埋芡後撈起備用。

3. 將菜及鹹肉加入飯中拌勻即可。

貼士

莖部比菜葉慢熟，所以要先煮，可使菜葉保持嫩綠，外觀跟口感亦較好。

清炒蝦仁

時間：20分鐘
製作難度：★☆☆☆☆

味道清甜的蝦仁，以清炒形式烹調，入口爽滑彈牙，佐飯亦一流。

材料

淡水蝦600克、生粉適量、蒜米/薑米各2茶匙

醃料：蛋白1/2隻、鹽/糖各少許、生粉1茶匙

調味料：雞粉1茶匙、水少許、麻油/紹酒各2茶匙、生粉水適量

做法

1.淡水蝦去殼，以牙籤挑走腸，下生粉拌勻、冲水、抹亁備用。

2.蝦仁加醃料醃10分鐘備用。

3. 大火燒熱油，加入蝦仁走油，立即取出瀝亁油。

4. 中火燒熱油，下薑米及蒜米爆香，蝦仁回鑊，灒酒後下雞粉、水及麻油調味，然後轉大火兜勻，以生粉水埋芡後便可上碟。

貼士

蝦要爽滑彈牙，一定要抹亁水後，才做走油或汆水過程。而加蛋白，亦可令蝦肉更爽口及雪白。

菜膽獅子頭

時間：20分鐘
製作難度：★★☆☆☆

食獅子頭又怕太油，可加小棠菜墊底，吸收了油份，少了油膩，口感更好。

材料

豬肉碎（後蹄肉）500克、小棠菜適量、甘笋數片、薑數片、罐裝笋適量、鹽/糖/油各少許

肉丸調味料：生抽2湯匙、水1至2湯匙、糖/麻油/薑茸/五香粉各2茶匙、生粉1茶匙、鹽少許

芡汁：生抽11/2湯匙、老抽1/2湯匙、糖2茶匙、生粉水適量、水1杯

做法

1. 煲滾水加鹽、糖及油，下小棠菜灼熟後撈起備用。

2. 豬肉加入調味料，以同一方向攪拌打成膠。

3. 將（2）搓成肉丸。大火燒熱油轉中火，放入肉丸炸至金黃，瀝亁油備用。

4. 大火燒熱油轉小火，爆香薑片、笋及甘笋，加入肉丸、老抽、生抽、水同糖，慢火燴3至5分鐘至入味，最後以生粉水埋芡，淋在小棠菜上即可。

貼士

後蹄肉略肥，肉質打起來易起膠同彈牙，做肉丸最適合。
