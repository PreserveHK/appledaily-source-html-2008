度掂佢：日防夜防熱刺難防 - 華度

16歲少女遇害，兇徒事後把心一橫，以極駭人手法，企圖毀屍滅迹。少女失蹤至命案被揭發，前後不過一星期，警方甚至能火速破案，兼將案發經過詳細呈現，想必與兇徒一五一十和盤托出有關。
聽講，命案受害者越死得慘，案件反而越快被偵破。兇手擺脫唔到犯案過程所帶嚟嘅恐怖回憶，備受心魔折騰；死者怨氣重，冥冥中亦會有所安排，使其沉寃得雪，天網恢恢，疏而不漏。
老竇從小對華度訓言，防人之心不可無，害人之心更不可有；聽晚熱刺平手盤，唔會害你。
