雞粒翻炸　爆米花

見Queenie食極唔肥，好生羨慕。尤其當知道她今日要教整一些煎炸食物，更令我感好奇，她說：「炸食物最好炸兩次，第二次炸係將油份逼出嚟，冇咁油膩，而且更加脆口㗎！」終於真相大白，難怪她能保持瓜子口面啦！

記者：黃潔蓮
攝影：陳盛臣
示範：陳莉敏

Queenie

美麗廚娘Queenie，深信煮得一手好菜，也不一定會變成黃面婆，故她為烹調美食而出盡法寶，戴潛泳鏡切洋葱、炸食物時會頭戴浴帽，免頭髮被油煙熏……貪靚更愛煮。

椒鹽乳酪雞米花

（4人份量/40分鐘）

我最愛吃台灣小食鹽酥雞，當年還特地買地瓜粉來蘸炸，卻做不到雞肉嫩滑而外皮脆的效果。Queenie教路，雞肉先用乳酪醃過，才會令肉滑，而且要炸兩次，將油份逼出，才會脆口呢！

材料

雞柳10條、椒鹽少許、麵包糠50克、蛋2隻、油適量

醃料：乳酪2湯匙、低鹽生抽1湯匙、胡椒粉/糖/雞粉/生粉各1茶匙

做法

1. 雞肉切成丁粒，加醃料醃半個鐘。

2. 雞肉先蘸拂勻嘅蛋漿，然後蘸上麵包糠，放入油鍋用細火炸熟，撈起。

3. 再放回油鍋，用中火炸將油份逼出嚟，約15秒。撈起瀝亁油，撒上椒鹽即成。

貼士

醃雞肉除可用原味乳酪，還可用含有果肉的乳酪，會更香甜。

香辣肉鬆生菜包

（4人份量/40分鐘）

肉吃得太多會覺得膩，用清新的生菜包裹着，就可減退膩滯感。

材料

蝦米10隻、冬菇5隻、西生菜1個、辣椒2隻（切碎）、脢頭肉1斤、馬蹄3粒、豆角粒100克、蒜茸2湯匙

調味料：魚露2茶匙、印尼甜豉油2湯匙、糖1茶匙、胡椒粉少許

醃料：胡椒粉/生粉各1茶匙、糖1湯匙、生抽3湯匙

做法

1.蝦米浸軟後切幼粒；冬菇去蒂，浸軟後揸亁水，切粒備用；豆角粒汆水，撈起瀝亁備用；生菜切半，剪成碗形備用。

2. 脢頭肉切去肥膏再切件，然後用兩把菜刀剁成肉醬，加醃料醃30分鐘備用。

3. 燒熱鑊後加入油，轉細火爆香蒜茸，加入蝦米粒、冬菇粒及馬蹄粒炒勻後盛起。燒熱鑊後，加入脢頭肉醬炒香至八成熟，加豆角粒一齊炒，炒到豬肉熟透，將冬菇、蝦米及馬蹄回鑊，加入調味料，再放辣椒碎炒勻後盛起。

4. 生菜鋪碟上，載入肉碎材料，撒上葱花及芫荽，捲起即可食用。

貼士

肉碎最好自己剁，會更有嚼勁。

炸南瓜春卷

（4人份量/30分鐘）

有時生菜食完，肉鬆還剩下一大碟，Queenie說可以加南瓜包春卷。

材料

日本南瓜1/2個、春卷皮1包、粉絲1紮、炒肉鬆200克、油適量、生芝麻3湯匙、蛋白1隻、蛋白水2湯匙

汁料：喼汁3湯匙、橙果醬2湯匙

做法

1.南瓜切去硬皮，隔水蒸20分鐘至熟，切成丁粒，拌入炒肉鬆備用。

2. 粉絲浸軟，瀝亁水後剪碎。燒熱鑊，用細火炒南瓜粒、粉絲及肉碎，炒到亁身做成餡料，待凉備用。

3. 春卷皮置室溫半個鐘解凍，鋪平於砧板上，放入大約2湯匙餡料，捲好後搽少少蛋白於封口貼實。

4. 燒熱一鍋油，春卷一邊掃上蛋白水，然後蘸上芝麻，有芝麻的一邊放入油鍋，用中火炸到浮起，撈起瀝亁油。

5. 加果醬及喼汁進牛奶煲，用細火煮到出煙後熄火，食春卷時蘸個汁即成！

貼士

內裏的餡料已熟，不用炸太耐，只炸到表皮金黃即可。芝麻較易搶火，故要留意切勿炸燶。

黑糖地瓜咖喱汁

（4人份量/40分鐘）

咖喱汁本身會好稠，Queenie在咖喱汁中加入寒天（即大菜），令到汁醬會凝固如啫喱狀，增加纖維素。

材料

黑糖2湯匙、咖喱粉1湯匙、洋葱2個、魚露/寒天各少許、紅蘿蔔1/2條、日本地瓜2個、雞湯1/2杯、老抽1/2湯匙

做法

1.洋葱先放入微波爐加熱5至10秒，就唔會刺激雙眼流眼淚，然後去皮、切絲；紅蘿蔔切粒；地瓜切件備用。

2. 地瓜放入滾油中，炸至表面微黃，撈起瀝亁油備用。燒熱鑊加少少油，炒香洋葱，然後放入咖喱粉，等到啲香味滲出，加埋雞湯、紅蘿蔔粒及地瓜炒勻。

3. 如見水收亁，可加入適量的水。煮到汁略為杰身，加埋老抽、黑糖及魚露煮到滾，最後加寒天煮勻，就可以食得！

貼士

地瓜先用熱水烚一烚，炸好後會更脆。
