一個人一個故事
薩爾科齊次子成少女偶像

高大英俊、一頭長長金髮，散發一股懾人魅力，他不是荷李活明星，但比很多明星更受少女歡迎。他是法國總統薩爾科齊（NicolasSarkozy）的21歲兒子──讓．薩爾科齊（JeanSarkozy）。
讓．薩爾科齊年輕有為，剛當選黨內要職，現在成為萬千少女和同志偶像，有粉絲更高呼：我要嫁給他。

21歲已是政壇新星

讓是薩爾科齊和首任妻子所生的二子，現仍在大學讀法律，三個月前初踏政壇，代表執政人民運動聯盟競選巴黎郊區諾伊利市議員，一舉當選，日前又獲選人民運動聯盟上塞納省地方議會主席，保住父親政治根據地。
21歲已成為政治新星，比父親直至28歲才在政壇嶄露頭角，讓顯然青出於藍。火速上位，政敵質疑是薩爾科齊刻意扶拔，讓對此不以為然：「人們可對我的年齡和名字說三道四，我只管做好工作。」
讓遺傳了風流父親的俊臉，魅力比老父猶有過之，大批少女和同性戀者像追捧年輕偶像一樣崇拜他，為他成立粉絲網站，有人甚至在Facebook開設「我要嫁給讓．薩爾科齊」的群組。
可惜，萬千少女要心碎了。在剛過去的周日，讓已跟青梅竹馬的高中女友塞邦（JessicaSebaoun）訂婚。
